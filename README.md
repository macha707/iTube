# iTube

### YouTube Downloader for Android

#### [\[Download APK\]](https://github.com/nhcodes/iTube/releases)

#### Features

- Supports Videos, Playlists, Shorts, ...
- Fast Downloads
- Integrated Media Player & Playlists
- Share from YouTube
- Ignores Ads
- Supports 99% of devices (Android 5+)
- 100% Free & No Ads

#### Technologies

- Kotlin
- AndroidX
- Jetpack Compose
- Material 3
